# AR-BirdShooter 
This is a mini project I did to test AR Foundation in unity. 
Shoot Alien birds invading our world.

#Features
1. Virtual Birds spawn in the real world
2. Aim & Shoot
3. SoundFX
4. Blood Splatter ParticleFX
